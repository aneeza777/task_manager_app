# 📋 Task Manager App (Flutter + Firebase)

A simple and professional Task Manager mobile application built using **Flutter** and **Firebase Firestore**.

This app was developed as part of a 3-week Flutter learning and project assignment.

---

## 🚀 Features

- Splash Screen
- Login Screen with form validation
- Navigation between screens
- Add, delete, and mark tasks as complete
- Real-time cloud sync using Firebase Firestore
- Professional UI with Cards, Icons, and AppBar
- Clean folder structure and readable code

---

## 🔥 Firebase Integration

This app uses **Cloud Firestore** to store tasks in real time.

- Tasks are saved in the cloud
- Data syncs instantly across devices
- No local storage used
